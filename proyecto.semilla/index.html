Que es el proyecto Semila

El proyecto semilla fue creado pensando en los agricultores y productores de toda la region donde conocemos la problematica que muchos emprendedores tienen que afrontar. Seguro te há pasado que despues de todo el esfuerzo que conlleva cosechar sus productos tienen dificultades para comercializarlos. Es por eso que fue creado este pequeño espacio web denominado "proyecto semilla", donde podras encontrar videos introductorios para comenzar a aprender!!! con videos cortos pero efectivos que manejan la teoria sobre que es "Emprender", que es "comercializar" y como podemos usar las diferentes estrategias de Marketin normal y Marketing virtual. Tambien podras tener acceso a una pequeña cartera de posibles clientes de todo bolivia seleccionadas especialmente para ti. Te desea mucho exitos...Proyecto semilla
